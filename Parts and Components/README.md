# Bauteile

In diesem Ordner sind alle Bauteile, die man braucht, um das Stethoskop zu bauen. Dabei werden die Bauteile in drei Kategorien unterteilt. Es wird unterschieden zwischen den elektrischen Bestandteilen, den Bauteilen die zum 3D-Drucken konstruiert wurden und allen kleinen Zusatzteilen, die keiner eigenen Gruppe bedürfen.
